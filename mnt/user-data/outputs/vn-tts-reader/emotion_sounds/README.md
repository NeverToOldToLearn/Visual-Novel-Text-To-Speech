# emotion_sounds/

Place your custom `.wav` (or `.mp3`) sound files here.  
The reader will play them automatically when it encounters matching emotion tags in dialogue.

## Expected filenames

| Emotion tags in text                        | File to provide         |
|---------------------------------------------|-------------------------|
| `*laughs*`, `*laugh*`                       | `laughs.wav`            |
| `*chuckles*`, `*chuckle*`                   | `chuckles.wav`          |
| `*giggles*`, `*giggle*`                     | `giggles.wav`           |
| `*sighs*`, `*sigh*`                         | `sigh.wav`              |
| `*groans*`, `*groan*`                       | `groan.wav`             |
| `*gasps*`, `*gasp*`                         | `gasp.wav`              |
| `*yawns*`, `*yawn*`                         | `yawn.wav`              |
| `*moans*`, `*moan*`                         | `soft_moan.wav`         |
| `*coughs*`, `*cough*`                       | `coughs.wav`            |
| `*sniffles*`, `*sniffle*`                   | `sniffle.wav`           |
| `*pants*`, `*panting*`                      | `pants.wav`             |
| `*slurps*`, `*slurp*`                       | `slurp.wav`             |
| `hmm`, `hmmm`                               | `hmm.wav`               |
| `mmmm`                                      | `mmmm.wav`              |
| `ah`, `ahh`                                 | `ah.wav`                |
| `mhmm`                                      | `mhmm.wav`              |
| `*Soft Moan*`                               | `soft_moan.wav`         |
| loud `AAH` variants                         | `yell.wav`              |

## Notes
- Both `.wav` and `.mp3` are supported.
- Files are optional — missing files are silently skipped.
- Keep samples short (< 3 seconds) for the best immersion.
- You can use AI voice generators, Freesound.org, or record your own.
